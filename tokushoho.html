支払方法

Apple ID / App Store の決済方法によります。

支払時期

  • 買い切り：購入確定時に請求されます。
  • 自動更新サブスクリプション：購入確定時および各更新時に請求されます。現在の期間終了の24時間以上前に解約しない限り、自動更新されます。
  • 年額プラン：App Storeがintroductory offerの対象と判定した利用者のみ7日間無料です。対象者が無料期間終了の24時間以上前に解約しない場合、年額料金へ自動移行します。対象外の利用者は購入画面に表示された条件で購入時から課金されます。

提供時期

購入手続き完了後、Apple / RevenueCat の購入状態反映が完了し次第、対象機能を利用できます。通信状態やストア側の反映状況により、利用開始まで時間がかかる場合があります。

解約方法

自動更新サブスクリプションは、利用者自身が以下から解約できます。

iPhone の設定 → Apple ID → サブスクリプション → Krites

アプリ削除やアカウント削除だけでは、Apple のサブスクリプションは解約されません。

返品・キャンセル・返金

デジタルコンテンツの性質上、購入確定後の返品・キャンセルはできません。App Store 経由の購入に関する返金は、Apple の返金制度に基づき Apple が判断・処理します。返金を希望する場合は reportaproblem.apple.com から申請してください。

動作環境

本アプリは iOS アプリおよび Mac / Chrome 拡張機能として提供されます。対応 OS、ブラウザ、端末条件は App Store / Chrome Web Store の表示およびアプリ内案内に従います。

クロスデバイス機能は、Chrome が起動し、拡張機能が有効で、同期が完了している場合に動作します。同期には最大で約5分かかる場合があります。Chrome 以外のブラウザ、拡張が無効な状態、通常設定でのシークレットウィンドウは対象外です。

特別な販売条件

本アプリは、利用者本人の自己管理を目的とする集中補助アプリです。保護者管理、従業員管理、MDM、医療、診断、治療、法的な監視を目的としたサービスではありません。OS 権限、通信状態、利用者自身による権限解除、アンインストール、再インストール等により、制限が働かない場合があります。
