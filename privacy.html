Krites（以下「本アプリ」）は、利用者本人が自分の集中時間を管理するためのアプリです。本ポリシーは、本アプリが取り扱う情報、利用目的、保存、共有、削除について説明します。

1. 基本方針

本アプリは、利用者が自分で決めた集中ルールを実行しやすくするために、必要な最小限の情報だけを扱います。広告配信、第三者トラッキング、データ販売は行いません。

本アプリは、利用者が何を読んだか、何を入力したか、何を話したかを収集するサービスではありません。

2. 取得・保存する情報

本アプリは、機能提供に必要な情報を Google Firebase（Cloud Firestore / Authentication、東京 asia-northeast1）に保存します。

3. Family Controls について

本アプリは、iOS の制限対象選択に Apple の Screen Time / Family Controls を利用します。Family Controls の選択は Apple により不透明なデータとして扱われ、本アプリはその中身を人間が読めるアプリ名・サイト名として取得できません。

Family Controls は、利用者本人の自己管理のために使います。他人の端末を管理する目的ではありません。

4. Mac / Chrome 拡張について

Mac / Chrome 拡張は、利用者が登録したサイトや URL ルールを Chrome 上で判定し、対象に一致したページを集中画面へ切り替えます。

  • URL の照合はブラウザ内で行います。
  • 閲覧履歴、ページ内容、フォーム入力、パスワード、メッセージ内容は送信・保存しません。
  • ブロックが発動した場合は、対象ラベル、日時、端末種別など、本人の記録表示に必要な情報だけを保存します。
  • Chrome が起動していない、拡張が無効、同期未完了、Chrome 以外のブラウザ、通常設定でのシークレットウィンドウでは動作しません。
  • 同期には最大で約5分かかる場合があります。

5. 取得しない情報

本アプリは、以下を取得しません。

  • 閲覧履歴そのもの
  • パスワード、フォーム入力内容
  • メッセージ、メール、通話、音声の内容
  • 位置情報
  • 連絡先、写真、マイク、カメラ
  • 広告識別子を使ったトラッキング情報

6. 外部サービス

本アプリは、以下の外部サービスを利用します。

  • Google Firebase: 認証、Firestore 保存、同期
  • RevenueCat: サブスクリプション・買い切りの購入状態管理
  • Apple App Store: 決済、購入復元（購入は利用者の Apple ID に紐づいて処理されます）
  • Chrome Web Store / Chrome Extension APIs: Mac / Chrome 拡張の配布とブラウザ内制御

本アプリは、広告SDK、第三者広告トラッキング、データ販売を行いません。

7. 保存期間と削除

データは、アカウントが有効である間、機能提供に必要な範囲で保存されます。

アプリ内のアカウント削除から、Krites 側に保存されたデータを削除できます。削除対象には、スケジュール、設定、保存パターン、今すぐ集中、常に許可する設定、ハードモード状態、イベント、統計、接続状態、購入状態の記録、ローカル状態が含まれます。

アカウント削除後に本アプリを再度利用し始めても、以前の Krites データは復元されません（新しい匿名アカウントとして開始します）。

ただし、Apple のサブスクリプション解約は別手続きです。アカウント削除やアプリ削除だけでは、自動更新は停止しません。

8. 国外からの利用

本アプリの Firebase データは、日本（東京 asia-northeast1）に保存されます。日本国外から利用する場合、データが日本に転送・保存されることに同意したものとみなされます。

9. 子どものプライバシー

本アプリは一般向けであり、13歳未満の子どもから意図的に個人情報を収集しません。

10. ポリシーの変更

本ポリシーを変更した場合は、本ページの最終更新日を更新します。重要な変更がある場合は、アプリ内または公開ページで通知します。

English Summary

Krites is a self-management focus app. No sign-in is required; Krites creates an anonymous account automatically. It stores only the information needed to provide the service: an anonymous account ID, blocking settings, schedules, selected-target data, block/unblock records, stats, Mac extension sync status, pairing information, and purchase entitlement status.

Krites uses Apple's Screen Time / Family Controls for iOS self-management. Family Controls selections are opaque data; Krites cannot translate them into human-readable app or website names.

The Mac / Chrome extension checks matching URLs locally in the browser and switches matching pages to a focus screen. It does not send or store browsing history, page content, form input, passwords, or messages. It works only when Chrome is running, the extension is enabled, and sync has completed; sync may take about 5 minutes.

Krites uses Firebase, RevenueCat, Apple App Store, and Chrome Extension APIs. It does not use ad SDKs, third-party ad tracking, or data selling.

Account deletion deletes the Krites account data, but it does not cancel an Apple subscription. Subscriptions must be canceled in Apple ID subscription settings.
